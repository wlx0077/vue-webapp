# webapp-qqmusic

## 项目初始化
```
npm install
cd webapp-qqmusic
```
### 下载并启动api接口
```
git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git
npm install
```

### 启动项目
```
npm run serve
```

### 构建项目
```
npm run build
```

### 测试
```
npm run test
```

### lint检查
```
npm run lint
```
